# Morphological Image Processing for Procedural Dungeon Generation

## Libraries used
1.  SDL2
2.  boost

## Prerequisites to build
1.  clang++
  a.  newer version required. C++x11 standards needed for some parts of the project
2.  cmake
3.  boost
4.  SDL2
5.  SDL2_image

## How to build
```
mkdir build
cd $_
cmake ..
make
```
